Task Tracker adalah aplikasi web sederhana untuk mencatat, mengelola, dan memantau tugas sehari-hari.
Aplikasi ini menggunakan tampilan **sticky notes**, mendukung **penandaan penting**, **penandaan selesai**,
**filter tugas**, serta **sistem pengingat otomatis** berdasarkan deadline. Semua data disimpan menggunakan
**localStorage**, sehingga tetap tersimpan meskipun halaman ditutup.
